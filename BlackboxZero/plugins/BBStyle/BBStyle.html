bbStyle 4.0
description
bbStyle is a Blackbox for Windows plugin that applies a random style to Blackbox on a set timer and|or on startup.

bbStyle 4.0 requires any Blackbox for Windows flavour issued since 2005.

initial setup
  • Use the load plugin menus provided in your flavour of bb4win and browse to the location of the bbStyle dll
  • OR add the relative path of the plugin to the BB configuration file plugins.rc
    • located in the Blackbox directory
    • OR accessed by Win+P with bbKeys)
    For example, if Blackbox is installed in c:\Blackbox, and bbStyle.dll is located in c:\Blackbox\plugins\bbStyle\, you should add the line...

    plugins\bbStyle\bbStyle.dll ...to plugins.rc.

    Simple, eh? :)

    how to use bbStyle...

    Mouse Input

    Once the plugin is loaded, several things can be done with a simple click on the bbStyle "button":

    Left Click  - applies a randomly selected style to Blackbox for Windows.
    LeftClick + Ctrl  - this lets you drag the bbStyle around
    LeftClick + Shift  - toggles display of styleInfo
    Middle Click  - opens your currently selected fileTree of Styles
    RightClick  - access the plugin Menu
    RightClick + Ctrl  - access the plugin Menu
    RightClick + Shift  - toggle in & out of the slit

    The Menu

    The plugin menu let's you change or select various things, and most should be self-explanatory.

    If bbColor3dc or bbColor_3dc2 is also loaded then a separate subMenu which provides support for all the bbColor3dc options - as well as (for bbColor_3dc2) access to its RC file & documentation.

    Keyboard Input

    Make bbStyle the focused window & it can accept keyboard input. The keys you can press to access a few functions quickly are below.
    Note: to re-position you need NumLock on.

    Numpad Keys 1-9, except 5  Set bbStyle position (7 is TopLeft, 3 BottomRight, etc.)
    Numpad Back  Access the Browse function for your Styles fileTree (not xoblite)
    Numpad Add  Toggle alwaysOnTop
    Escape  Toggle whether the changeTimer is running

    Miscellaneous Notes

    Moving bbStyle: You cannot drag bbStyle off the virtual/viewable screen area manually. To move bbStyle to a different monitor, Ctrl+LeftClick and drag the plugin into the prospective monitor's screen area. Alternatively, set Use Full Screen on the Placement subMenu to True & use the Placement subMenu.

    bbStyle.rc settings

    Default settings are:

    bbStyle.stylePath: <bb4win path>\styles\  Path to styleFiles - searched recursively
    bbStyle.X: 775  X position
    bbStyle.Y: 10  Y position
    bbStyle.onTop: true  Always on top?
    bbStyle.snap: 20  Distance before window locks to screen edge
    bbStyle.togglePlugins: true  Toggle visibility with all other plugins
    bbStyle.showStyleInfo: true  Show the styleName & Author?
    bbStyle.fullScreen: false  Position according to multi-monitors?
    bbStyle.useSlit: false  Hold bbStyle inside the Slit?
    bbStyle.timerOn: false  Enable timer?
    bbStyle.changeTime: 5  Number of minutes between style changes
    bbStyle.changeOnStart: false  Apply a random style on bb4win startup?
    bbStyle.chance: false  Change styles randomly? (Default is sequentially)

    bbStyle.alphaValue:   Value of transparency (0-8) -> default is Menu.alpha

    bbStyle bro@ms

    @bbShowPlugins  Standard BB bro@m to show all plugins
    @bbHidePlugins  Standard BB bro@m to hide all plugins

    @bbStyle.Random  Select a random style and apply it
    @bbStyle.EditRC  Open the current style with your preferred editor
    @bbStyle.About  Display message box about the plugin

    Value & True|False bro@ms can also be set for most of the bbStyle entries:
    @bbStyle.snap   @bbStyle.onTop   @bbStyle.snap   @bbStyle.alphaValue   @bbStyle.togglePlugins   @bbStyle.showStyleInfo   @bbStyle.useSlit   @bbStyle.timerOn   @bbStyle.changeTime   @bbStyle.changeOnStart  

    POSITIONING: These set the position of the Style window to that specified, according to bbStyle.fullScreen (false = current monitor only; true = full screen)
    @bbStyle.TopLeft   @bbStyle.TopCenter   @bbStyle.TopRight   @bbStyle.CenterLeft   @bbStyle.CenterRight   @bbStyle.BottomLeft   @bbStyle.BottomCenter   @bbStyle.BottomRight
